# AI CV Analyzer

This project is an AI-powered CV analysis system built with ASP.NET MVC.

## 🚀 Features
- Analyze CV text using AI
- Upload PDF CV and extract content
- Get feedback on strengths, weaknesses, and suggestions
- CV scoring system
- View analysis history

## 🛠 Technologies
- ASP.NET MVC
- Entity Framework
- MSSQL
- OpenAI API
- iText7 (PDF parsing)

## 📌 How it works
User enters CV text or uploads a PDF. The system sends it to OpenAI API and returns structured analysis including strengths, weaknesses, and suggestions.

## ⚠️ Note
API key is not included. You need to add your own OpenAI API key in the project.
